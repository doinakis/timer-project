# Timer Project
In addition to the revision of the [prods-cons](https://github.com/doinakis/Real-Time-Embedded-Systems) this is a follow up project. It is intended to be use with a raspberry pi and it is implementing a simple timer that periodically pushes functions to a fifo queue. It is implemented using pthreads, every timer is a producer thread and every consumer is a worker thread that executes the functions that the timers push to the fifo queue. The number of consumers and the length of the fifo queue can be assigned at the globaldef.h file. The timer.h timer.c and main.c files should have everything you need to understand the code and adjust it to your needs. The [cross-compiler](https://github.com/abhiTronix/raspberry-pi-cross-compilers) used can be found in the above link.
